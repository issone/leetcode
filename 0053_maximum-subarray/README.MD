# [53. 最大子序和](https://leetcode-cn.com/problems/maximum-subarray/)


## 题目

给出一个区间的集合，请合并所有重叠的区间。
 

示例1：
 
```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。

```

示例2：
 
```
输入：nums = [1]
输出：1

```

示例2：
 
```
输入：nums = [1]
输出：1

```

示例3：
 
```
输入：nums = [0]
输出：0

```

示例4：
 
```
输入：nums = [-1]
输出：-1

```

示例5：
 
```
输入：nums = [-100000]
输出：-100000

```
提示：

- 1 <= nums.length <= 3 * 10^4
- -10^5 <= nums[i] <= 10^5
